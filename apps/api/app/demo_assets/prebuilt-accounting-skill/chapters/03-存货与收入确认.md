# 第3章 存货与收入确认

## 核心概念
- **存货**: 为销售或生产持有的资产
- **发出存货计价**: 确定成本结转方法
- **可变现净值**: 存货减值判断基础
- **收入确认**: 在控制权转移时确认
- **履约义务**: 合同中承诺的可区分商品或服务

## 公式与计算口径
- **期末存货成本**: `期初存货 + 本期购入 - 本期发出`
- **存货跌价准备**: `成本 - 可变现净值（若为正）`
- **营业利润基础关系**: `收入 - 成本 - 税金及附加 - 期间费用`

## 方法/流程
1. 判断存货取得、发出与期末状态
2. 选择先进先出或加权平均等计价方法
3. 比较成本与可变现净值，判断是否减值
4. 收入按合同履约义务完成情况确认

## 例题
### 例 3.1: 存货跌价准备
- **已知**: 某批商品成本 120000 元，可变现净值 108000 元
- **求**: 应计提的跌价准备
- **解**:
  1. 跌价额 = 120000 - 108000 = 12000
  2. 计提后存货按 108000 元列示
- **结论**: 应计提存货跌价准备 12000 元

## 易混点 / 边界条件
- **收入确认**: 关注控制权，不只看收款
- **存货减值**: 跌价准备可以转回，但不得超过原已计提金额
