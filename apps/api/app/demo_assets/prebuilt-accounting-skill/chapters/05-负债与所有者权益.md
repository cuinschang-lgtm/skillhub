# 第5章 负债与所有者权益

## 核心概念
- **流动负债**: 一年内需偿还的债务
- **非流动负债**: 超过一年偿还的债务
- **所有者权益**: 资产扣除负债后的剩余权益
- **实收资本**: 投资者投入资本
- **留存收益**: 企业累计未分配利润

## 公式与计算口径
- **资产负债率**: `负债总额 / 资产总额`
- **权益乘数**: `资产总额 / 所有者权益总额`
- **未分配利润期末数**: `期初未分配利润 + 本期净利润 - 本期分配`

## 方法/流程
1. 区分短期借款、应付票据、应付债券等负债类型
2. 根据合同条款确认利息与偿付义务
3. 区分实收资本、资本公积、盈余公积和未分配利润
4. 结合报表分析企业杠杆水平

## 例题
### 例 5.1: 资产负债率计算
- **已知**: 资产总额 2000000 元，负债总额 800000 元
- **求**: 资产负债率
- **解**:
  1. 资产负债率 = 800000 / 2000000 = 40%
- **结论**: 企业资产负债率为 40%

## 易混点 / 边界条件
- **资本公积**: 不等于企业经营利润
- **短期负债**: 到期时间短，影响流动性判断
