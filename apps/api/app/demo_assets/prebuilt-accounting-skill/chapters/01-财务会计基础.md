# 第1章 财务会计基础

## 核心概念
- **财务会计目标**: 提供决策有用信息
- **可靠性**: 信息真实可验证
- **相关性**: 信息影响决策判断
- **会计确认**: 判断何时入账
- **会计计量**: 确定金额属性

## 公式与计算口径
- **会计恒等式**: `资产 = 负债 + 所有者权益` — 复式记账基础
- **期间利润**: `利润 = 收入 - 费用` — 衡量经营成果

## 方法/流程
1. 识别经济业务对应的会计要素
2. 判断是否满足确认条件
3. 选择合适计量属性
4. 形成会计分录并进入账簿系统

## 例题
### 例 1.1: 业务要素识别
- **已知**: 企业购入设备，银行转账支付 100000 元
- **求**: 判断涉及的会计要素
- **解**:
  1. 设备增加，属于资产增加 100000
  2. 银行存款减少，属于资产减少 100000
- **结论**: 同为资产项目内部变动，不影响权益总额

## 易混点 / 边界条件
- **确认**: 先判断是否应入账，再判断金额
- **计量**: 不是所有项目都长期按历史成本反映
