# 第6章 财务报表与分析

## 核心概念
- **资产负债表**: 反映某时点财务状况
- **利润表**: 反映某期间经营成果
- **现金流量表**: 反映现金流入流出
- **附注**: 补充解释重要会计信息
- **比率分析**: 用指标评价经营与财务状况

## 公式与计算口径
- **流动比率**: `流动资产 / 流动负债`
- **速动比率**: `(流动资产 - 存货) / 流动负债`
- **净利润率**: `净利润 / 营业收入`

## 方法/流程
1. 先看资产负债表判断规模与结构
2. 再看利润表判断盈利能力
3. 再看现金流量表验证利润质量
4. 结合附注解释异常波动与会计政策

## 例题
### 例 6.1: 流动比率计算
- **已知**: 流动资产 900000 元，流动负债 600000 元
- **求**: 流动比率
- **解**:
  1. 流动比率 = 900000 / 600000 = 1.5
- **结论**: 企业流动比率为 1.5

## 易混点 / 边界条件
- **盈利能力**: 不只看利润，还要结合现金流
- **报表分析**: 需要横向、纵向与趋势结合
