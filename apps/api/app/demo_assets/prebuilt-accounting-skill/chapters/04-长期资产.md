# 第4章 长期资产

## 核心概念
- **固定资产**: 使用寿命超过一年有形资产
- **折旧**: 按受益期分摊固定资产成本
- **无形资产**: 无实物形态的可辨认资产
- **摊销**: 无形资产成本系统分配
- **减值**: 资产可收回金额低于账面价值

## 公式与计算口径
- **年折旧额**: `(原值 - 预计净残值) / 使用年限`
- **账面价值**: `原值 - 累计折旧 - 减值准备`
- **无形资产摊销额**: `可摊销金额 / 受益年限`

## 方法/流程
1. 判断资产是否满足资本化条件
2. 区分固定资产与无形资产
3. 选择折旧或摊销方法并按期计提
4. 期末判断是否存在减值迹象

## 例题
### 例 4.1: 直线法折旧
- **已知**: 设备原值 300000 元，预计净残值 15000 元，使用年限 5 年
- **求**: 年折旧额
- **解**:
  1. 可折旧金额 = 300000 - 15000 = 285000
  2. 年折旧额 = 285000 / 5 = 57000
- **结论**: 每年计提折旧 57000 元

## 易混点 / 边界条件
- **维修支出**: 日常维修通常费用化
- **资本化**: 提高未来经济利益才可能资本化
