# 中级财务会计演示版

本 skill 是一份为演示场景预制的《中级财务会计》知识包，重点展示“教材如何被编译成可以问答的知识能力”。

## 使用流程（必读）

1. 如果用户问“这本书主要讲什么”，优先加载 `chapters/00-课程总览.md`
2. 如果用户问具体知识点，按下方章节速查表选择最相关的 1-2 章
3. 先基于章节内容回答，再补充课堂常见场景与应用
4. 若问题跨章，最多同时引用 2 个章节，保持回答简洁

## 章节速查表

| 章节 | 关键词（精确命中） | 文件 |
|------|------|------|
| 课程总览 | 会计要素、财务报表、课程结构、学习路径 | `chapters/00-课程总览.md` |
| 财务会计基础 | 会计目标、信息质量、确认、计量、会计要素 | `chapters/01-财务会计基础.md` |
| 货币资金与应收项目 | 现金、银行存款、应收账款、坏账准备 | `chapters/02-货币资金与应收项目.md` |
| 存货与收入确认 | 发出存货计价、可变现净值、收入确认、合同履约 | `chapters/03-存货与收入确认.md` |
| 长期资产 | 固定资产、折旧、无形资产、摊销、减值 | `chapters/04-长期资产.md` |
| 负债与所有者权益 | 短期借款、应付债券、实收资本、留存收益 | `chapters/05-负债与所有者权益.md` |
| 财务报表与分析 | 资产负债表、利润表、现金流量表、报表分析 | `chapters/06-财务报表与分析.md` |

## 反例

❌ 用户问“课程整体讲什么”时，只引用某一章局部概念
❌ 用户问“坏账准备怎么做”时，不区分应收项目与存货跌价
✅ 用户问“这本书主要讲什么”时，先概括课程结构，再列核心模块
