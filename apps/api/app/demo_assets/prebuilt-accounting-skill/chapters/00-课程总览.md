# 课程总览：中级财务会计

## 核心概念
- **课程定位**: 财务会计核心专业课
- **知识主线**: 确认、计量、记录、报告
- **会计对象**: 资产、负债、所有者权益
- **成果输出**: 财务报表与分析

## 公式与计算口径
- **会计恒等式**: `资产 = 负债 + 所有者权益` — 全书逻辑主线
- **利润关系**: `利润 = 收入 - 费用` — 经营成果基础口径

## 方法/流程
1. 识别经济业务的会计要素归属
2. 判断确认时点与计量属性
3. 编制会计分录并归集到报表项目
4. 结合报表解释企业经营状况

## 例题
### 例 0.1: 课程结构概括
- **已知**: 学生要在 2 分钟内理解课程范围
- **求**: 给出课程总体框架
- **解**:
  1. 先讲六大要素与会计恒等式
  2. 再讲资产、负债、权益、收入费用利润
  3. 最后落到三大报表与分析应用
- **结论**: 课程围绕“业务处理到报表输出”展开

## 易混点 / 边界条件
- **总览**: 强调课程结构，不替代分章细节
- **报表分析**: 依赖前面各章会计处理结果

## 关联
- 前置: 会计学原理、基础会计
- 应用: 财务报表分析、审计、税务与管理决策
