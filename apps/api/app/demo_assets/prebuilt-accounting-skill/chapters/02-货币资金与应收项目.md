# 第2章 货币资金与应收项目

## 核心概念
- **货币资金**: 现金、银行存款、其他货币资金
- **应收账款**: 因赊销形成的债权
- **坏账准备**: 应收项目减值准备
- **应收票据**: 以票据形式存在的债权
- **预付账款**: 采购前先付款形成的债权

## 公式与计算口径
- **应收账款净额**: `应收账款余额 - 坏账准备` — 资产负债表列示口径
- **坏账准备调整额**: `期末应有坏账准备 - 账上已有坏账准备`

## 方法/流程
1. 识别款项性质是货币资金还是应收项目
2. 判断是否需要计提坏账准备
3. 按净额口径列示在报表中
4. 对票据、预付款等项目单独分类反映

## 例题
### 例 2.1: 坏账准备调整
- **已知**: 期末应收账款 500000 元，按 5% 计提；账上已有坏账准备 18000 元
- **求**: 本期应补提坏账准备
- **解**:
  1. 期末应有坏账准备 = 500000 × 5% = 25000
  2. 本期补提 = 25000 - 18000 = 7000
- **结论**: 本期应补提坏账准备 7000 元

## 易混点 / 边界条件
- **坏账核销**: 核销不等于新增费用
- **预付账款**: 实质是债权，不是存货
